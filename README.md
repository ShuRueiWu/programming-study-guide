# 程式學習指南

給初學者的程式**概念 + 速查**整合單頁（繁體中文，技術名詞保留英文）。單一自包含 HTML，無外部相依、可離線開啟、可列印成 PDF。暖米白主題。

## 內容

- **觀念（22 章）**：程式如何執行、變數型別、控制流程、函式、資料結構選型、OOP、例外處理、模組/虛擬環境、解題流程、Debug、Git、測試、Big-O、搜尋/排序/遞迴、JavaScript、SQL、pandas、Docker、HTTP/API、進階資料結構
- **速查表（16 張）**：HTML、CSS、CSS 進階、Python、Python 進階、Python 常見錯誤、Python 標準庫、JavaScript、SQL、Regex、JSON/YAML、Markdown、Shell、醫療統計、Docker、Claude Code
- **互動計算器（3 個）**：診斷指標 2×2（含 95% Wilson CI）、療效/風險（RR/OR/ARR/NNT）、檢驗後機率（Fagan / 貝氏）

查表區有即時搜尋過濾框；整份可 `Cmd/Ctrl + P` 列印成滿版 PDF。
